- 🧑🏻‍🔬 Add another admins for you project 🧑🏻‍🔬

## 🔌 Setup 🔌

1. Clone the project:

```shell
git clone https://github.com/renattele/admin95
```

2. Copy (.sample.env)[.sample.env] to .env:

```shell
cp .sample.env .env
```

3. Change variables as described in (.sample.env)[.sample.env]
4. Run the project:

```shell
docker compose up -d
```
